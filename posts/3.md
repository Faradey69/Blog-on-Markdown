
# Асинхронный JavaScript: Promise и async/await

Асинхронность позволяет выполнять долгие операции (загрузка данных, таймеры) без блокировки основного потока.

## Колбэки (Callbacks)

```javascript
function loadData(callback) {
    setTimeout(() => {
        const data = { name: "Иван", age: 25 };
        callback(data);
    }, 1000);
}

loadData((result) => {
    console.log("Данные загружены:", result);
});
```
