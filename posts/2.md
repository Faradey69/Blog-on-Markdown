# Работа с DOM и событиями

DOM (Document Object Model) — это программный интерфейс, который представляет HTML-документ в виде дерева объектов. С помощью JavaScript мы можем изменять этот документ в реальном времени.

## Поиск элементов

Для работы с элементами страницы используются методы поиска:

```javascript
// Поиск по ID
const header = document.getElementById('main-header');

// Поиск по CSS-селектору
const firstButton = document.querySelector('.btn');

// Поиск всех элементов
const allLinks = document.querySelectorAll('a');
