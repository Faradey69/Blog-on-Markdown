# Введение в JavaScript

JavaScript — это язык программирования, который оживляет веб-страницы. Вместе с HTML и CSS он составляет основу современной веб-разработки.

## Переменные и типы данных

В JavaScript переменные объявляются с помощью ключевых слов `var`, `let` или `const`:

```javascript
let name = "Мир";
const PI = 3.14159;
var oldWay = "Устаревший способ";
```
